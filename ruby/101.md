## Symbols
>scalar value objects used as identifiers, mapping immutable strings to fixed internal values

## Why use symbols instead of strings?
>The immutable nature of symbols makes them very valuable in programming because mutable objects can cause bugs that are difficult to detect. Since symbols stay the same, using them helps to avoid this issue

## Hashes / Dictionaries
>A hash is a data structure used to store data in the form of UNIQUE key-value pairs
```
    a = { apple: 1, banana: 2}
    # Old ways
    # => Hash Rocket
    a = { :apple => 1, :banana => 3 }
    # Using strings instead of symbols
    a = { "apple" => 1, "banana" => 2 }
    # Again
    a = { apple: 1, banana: 2}
    # Values must be unique
    a = { apple: 1, banana: 2, apple: 3 }

    # Getting keys
    a.keys

    # Getting values
    a.values

    # Assign new value
    a[:melon] = 3

    # Acces with fetch
    a.fetch(:peach)
    a.fetch(:peach, 0)

    # Iterate a hash
    a.each do |key, value|
        puts key
        puts value
    end

    # Remove all values
    a.clear

    # Empty?
    a.empty?

    # Has a value
    a.has_value? :banana

    # Unique values counted
    a.length

    # Reject vs Reject!
    new_hash = a.reject { |b| b == :apple }
    new_hash.inspect

    puts a.reject! { |b| b == :apple }

    # Sort
    { b: 1, a: 2 }.sort

    # Sort BY
    { c: 3, b: 1, a: 2 }.sort_by(&:last)

    # Another way
    { c: 3, b: 1, a: 2 }.sort_by do |key| key.last end

    # To hash again
    { c: 3, b: 1, a: 2 }.sort_by(&:last).to_h

    # Merge two hashes
    defaults    = { a: 1, b: 2, c: 3 }
    preferences = { c: 4 }
    defaults.merge!(preferences)
```

## Classes on Ruby
```
# Define a class
class Box
    def initialize(w,h)
        # Instance variables
        @width, @height = w,h
    end

    def area
        return @width * @height
    end
end

box = Box.new 2,5
puts box.area

# Continue with the class
class Box
    # Accessor methods
    def getWidth
        @width
    end

    def getHeight
        @height
    end

    # Setter methods
    def setWidth(w)
        @width = w
    end

    def setHeight(h)
        @height = h
    end
end

# Using helpers
class Box
    attr_accessor :width, :height
    # attr_writer :width
    # attr_reader :height
end

box = Box.new 1,2
box.width
box.height
box.width = 2
box.height = 10
box.width
box.height
```

## Static or Class Methods

    @@only_giant = nil

end

## Inheritance
```
    class Animal
        attr_accessor :name
        def initialize animal_type
            @name = animal_type
        end

        def talk
            "Say something random"
        end
    end

    class Dog < Animal
        def bark
            puts "Wuf wuf!"
        end

        def talk
            bark()
        end
    end

    class Bird < Animal
        def squeak
            puts "Quick quick"
        end

        def talk
            squeak()
        end
    end

    class Hippo < Animal
    end
```
